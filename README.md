# Auth

Пример сервиса аутентификации примеры jwt c входом по электронной почте и паролю и отп коду (target sms) а так же примеры oauth2.0 (VK , Yandex, Google, Facebook, Telegram) 

полезный кусок кода чтоб скопипастить кода он мне будет нужен 



## Управление  проектом

1. Запукс линтера

```
make ruff

```

2. Запукс тестов

```
make tests

```
3. Запукс локально вне докера

```
make run_home

```

4. Создание миграций

```
make migration

```

5. Применить миграции

```
make upgrade
```

6. Запуск конейтнеров в фоне

```
make up_home_d

```

7. Запуск конетйнеров локально

```
make up_home

```

8.  Опустить контейнеры

```
make down

```

9. Создать виртуальное окружение и установить зависимости

```
make venv

```
