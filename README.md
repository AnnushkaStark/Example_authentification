# Auth

Пример сервиса аутентификации примеры jwt c входом по электронной почте и паролю и отп коду (target sms) а так же примеры oauth2.0 (VK , Yandex, Google, Facebook, Telegram) 

полезный кусок кода чтоб скопипастить кода он мне будет нужен 



## Управление  проектом

1. Запукс линтера

```
make ruff

```

2. Запукс локально вне докера

```
make run_home

```

3. Создание миграций

```
make migration

```

4. Применить миграции

```
make upgrade
```

5. Запуск конейтнеров в фоне

```
make up_home_d

```

6. Запуск конетйнеров локально

```
make up_home

```

7.  Опустить контейнеры

```
make down

```

8. Создать виртуальное окружение и установить зависимости

```
make venv

```
